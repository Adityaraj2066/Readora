# Readora 📚✨

A modern full-stack online bookstore and shopping cart application built using ASP.NET MVC, Entity Framework Core, and SQL Server.

## 🚀 Features

- User Authentication & Authorization
- Browse Books by Genre
- Add to Cart Functionality
- Checkout System
- Order Management
- Admin Dashboard
- Stock Management
- Book & Genre CRUD Operations
- Top Selling Books Report
- Responsive UI using Bootstrap

---

# 🛠 Tech Stack

- ASP.NET Core MVC (.NET)
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Identity Authentication

---

# 📂 Architecture

- MVC Architecture
- Repository Pattern
- Entity Framework ORM
