# Readora 📚✨

A modern full-stack online bookstore and shopping cart application built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

---

## 🚀 Features

* User Authentication & Authorization
* Browse Books by Genre
* Add to Cart Functionality
* Checkout System
* Order Management
* Admin Dashboard
* Stock Management
* Book & Genre CRUD Operations
* Top Selling Books Report
* Responsive UI using Bootstrap

---

## 🛠 Tech Stack

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Bootstrap 5
* Identity Authentication

---

## 🏗 Architecture

* MVC Architecture
* Repository Pattern
* Entity Framework ORM
* Razor Views

---

# ⚡ Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Adityaraj2066/Readora.git
```

---

## 2️⃣ Open Project

Open the solution file in Visual Studio:

```bash
BookShoppingCartMvc.sln
```

---

## 3️⃣ Configure Database

Update your connection string inside:

```bash
appsettings.json
```

Example:

```json
"ConnectionStrings": {
  "conn": "data source=YOUR_SERVER;initial catalog=ReadoraDB;integrated security=true;encrypt=false"
}
```

---

## 4️⃣ Run Migration

```bash
update-database
```

---

## 5️⃣ Run Project

Press:

```bash
Ctrl + F5
```
